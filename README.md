# TARANG – Instagram Clone

TARANG is a modern Instagram-style social app where users can share moments, connect with friends, and interact through chats, calls, and stories. It is built as a full-stack clone project to practice real-world social media features and UI/UX patterns.

## Features

-**birthday wishes on your birthday 

- **User Authentication**
  - Sign up / log in with unique accounts
  - Public and private profile modes
- **Profiles**
  - Editable profile with username, bio, and profile picture
  - Public/private profile visibility control
  - View other user profiles and follow/unfollow (if implemented)

- **Feed & Posts**
  - Instagram-like feed layout for posts
  - Support for images and captions (describe what you support)
  - Like, comment, and view post details (if implemented)

- **Stories**
  - Story bar similar to Instagram
  - Add and view stories
  - Temporary story display (you can note the duration if you implemented it)

- **Chats & Messaging**
  - One-to-one chat interface
  - **Message streaks** to show active ongoing conversations
  - **Calling feature through chats** (voice/video call entry point from the chat UI, describe how it works in your implementation)

## Status

TARANG is an active learning project. Expect ongoing changes as I iterate on features, fix bugs, and improve performance and responsiveness (especially mobile layout).
